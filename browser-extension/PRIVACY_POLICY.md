# DSG Transport Secure Login - Privacy Policy

**Last Updated:** December 2024

## Overview

DSG Transport Secure Login is a browser extension developed exclusively for DSG Transport LLC employees. This extension enables secure, automatic login to company tools without exposing credentials to users.

## Data Collection

### What We Collect
- **Extension ID**: Stored locally to enable communication with the DSG Transport dashboard
- **Temporary Login Data**: Encrypted credentials are temporarily stored in browser memory (auto-cleared after 2 minutes)

### What We Do NOT Collect
- Personal browsing history
- Passwords or credentials (these are encrypted and never visible)
- Personal information beyond what's needed for authentication
- Any data from non-DSG Transport websites

## Data Storage

- All data is stored **locally** in your browser using Chrome's secure storage API
- No data is transmitted to external servers except the DSG Transport backend for credential decryption
- Credentials are encrypted end-to-end and automatically cleared after use

## Data Sharing

- We do **NOT** sell, share, or transfer any user data to third parties
- Data is only transmitted between the extension and DSG Transport's secure servers

## Security

- All credential data is encrypted using Fernet encryption
- Credentials are never displayed to users
- Automatic data expiration after 2 minutes
- Extension only communicates with approved DSG Transport domains

## User Rights

- You can view stored extension data via `chrome://extensions`
- You can clear all extension data by removing and reinstalling the extension
- Contact your IT administrator for data access requests
