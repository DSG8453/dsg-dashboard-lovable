DSG Transport Secure Portal

Privacy Policy

Last Updated: January 2026

Overview

DSG Transport Secure Portal is a web application developed exclusively for DSG Transport LLC employees. This portal enables secure access to company tools through a backend-only authentication system where credentials are never exposed to the user's browser.

Data Collection

What We Collect

  • Authentication Data: Google Workspace SSO tokens for identity verification
  • Device Information: Browser fingerprint and device details for security purposes
  • Activity Logs: Records of tool access for audit and compliance

What We Do NOT Collect

  • Personal browsing history
  • Passwords or tool credentials (these are stored in Google Secret Manager and never sent to the frontend)
  • Personal information beyond what's needed for authentication
  • Any data from non-DSG Transport websites

Credential Security

  • All tool credentials are stored in Google Cloud Secret Manager
  • Credentials are never sent to the frontend or exposed in any browser response
  • Authentication happens entirely server-side through a secure gateway
  • Users can access tools without ever seeing the underlying credentials

Data Sharing

  • We do NOT sell, share, or transfer any user data to third parties
  • Data is only used to facilitate access to authorized company tools
  • All communication uses industry-standard TLS encryption

Security Measures

  • Google Workspace Single Sign-On (SSO) for authentication
  • Device approval system to prevent unauthorized access
  • Role-based access controls
  • All credentials stored in Google Secret Manager (not in application database)
  • Comprehensive activity logging for security audits

User Rights

  • Request a copy of your personal data
  • Request correction of inaccurate data
  • Request deletion of your data (subject to legal requirements)
  • Contact your IT administrator for any data-related requests

Contact

For privacy concerns, contact:

  • DSG Transport LLC
  • Email: info@dsgtransport.net
  • Website: portal.dsgtransport.net

Changes to This Policy

We may update this policy periodically. Changes will be communicated through company channels.
